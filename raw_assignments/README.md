# Raw Assignments

This directory preserves the iterative development steps of SHMACK, from initial SQL schema design through the final web application.

Each folder represents a progressive assignment submission:

- **Assignment 1–3**: Database schema design and SQL queries
- **Assignment 4–6**: Static HTML/CSS frontend prototypes
- **Assignment 7–9**: PHP backend with PDO database integration
- **Assignment 10**: Final MVP (source also available at project root in `public/`)

These are preserved for reference and to show the development progression.
